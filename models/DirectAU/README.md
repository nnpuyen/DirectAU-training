# DirectAU - Towards Representation Alignment and Uniformity in Collaborative Filtering

## Paper
**DirectAU: Towards Representation Alignment and Uniformity in Collaborative Filtering**  
KDD 2022  
Paper: https://arxiv.org/abs/2206.12811

## Official Repository
https://github.com/THUwangcy/DirectAU

## Core Idea
DirectAU directly optimizes two properties of the representation space:
- **Alignment**: Matching users should have similar representations to their preferred items.
- **Uniformity**: All representations should be uniformly distributed on the hypersphere.
